# Manual Web (UX acordada)

## Qué incluye
- Menú lateral colapsable con botón SIEMPRE visible (no se pierde la capacidad de reabrir).
- 3 pestañas: Instalación / Usuario / Operación.
- Buscador global: busca en TODO el contenido (no en el menú) y te lleva a la sección correcta (cambiando a la pestaña necesaria).
- Responsive (móvil/desktop).

## Cómo usar
1) Abrir `index.html` en un navegador.
2) Sustituir el texto de ejemplo por tu contenido real dentro de las 3 secciones `<section class="doc" data-doc="...">`.
